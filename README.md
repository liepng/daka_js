# 每日打卡 H5 应用

一个功能完善的 H5 Web 应用，用于记录个人每日打卡信息，支持文字、图片、视频和语音记录。

## 功能特性

### 1. 个人信息管理
- 设置姓名、性别、年龄
- 信息本地持久化存储
- 信息预览和修改

### 2. 每日打卡
- 文字记录（最多500字）
- 图片上传（最多9张）
- 视频上传（最多3个）
- 语音录制（最长10分钟）
- 支持预览和删除已选媒体

### 3. 打卡记录
- 列表展示所有打卡记录
- 查看记录详情
- 删除记录

### 4. 数据统计
- 总打卡次数统计
- 连续打卡天数统计
- 今日打卡状态

## 技术栈

- 原生 HTML/CSS/JavaScript
- LocalStorage 本地存储
- Node.js 静态文件服务（HTTPS）

## 使用说明

```bash
# 安装依赖
npm install

# 启动服务
npm start

# 浏览器访问 https://localhost:8443/h5/
```

## 项目结构

```
├── server.js              # HTTPS 静态服务器
├── package.json           # 项目依赖
├── h5/                    # H5 前端目录
│   ├── index.html         # 入口页面
│   ├── css/style.css      # 样式文件
│   └── js/bundle.js       # 应用代码（合并版）
└── images/                # 图标资源
```
